## 🎮 操作指南

### 基本操作
1. **复制URL**：点击工具栏复制按钮
2. **刷新页面**：点击刷新按钮
3. **全屏预览**：点击全屏按钮
4. **退出全屏**：点击右上角的退出按钮（仅在全屏模式下显示）
5. **旋转屏幕**：点击转屏按钮
6. **打开菜单**：点击菜单按钮或左滑

### 手势操作
- **左滑**：打开菜单
- **右滑**：关闭菜单
- **上滑**：隐藏工具栏
- **下滑**：显示工具栏

### 快捷键
- `ESC`：退出全屏/关闭菜单
- `F5`：刷新页面
- `F11`：切换全屏
- `Ctrl+C`：复制URL
- `Ctrl+M`：打开菜单
